Chapter 3: HTML Lists

Unordered Lists, Ordered Lists and Description Lists.

You can use lists like unordered lists, ordered lists and description lists to list out ingredients, steps or instructions, and much more! These lists are used to help organize data on a web page into a structured, easy-to-read format.

Example HTML Code:

<!DOCTYPE html>
<html lang="en">
<body>
    <p>This is a Unordered List</p>
<ul>
     <li>Beans</li>
    <li>Milk</li>
             <li>Cooking Oil</li>
			
			 
</ul>

<p>This is a Ordered List</p>
</ol>	
     <li>Preheat the Oven to 350 F</li>
    <li>Mix the ingredients</li>
             <li>Bake for 30 minutes</li>
</ol>	

    <p>This is a Description List</p>		
<dl>	
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>  
    <dt>CSS</dt>   
    <dd>Cascading Style Sheets</dd> 	
</body>
</html>
