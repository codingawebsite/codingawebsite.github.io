Chapter 4: Links

Hyperlinks.

You can use links to send users to another website or to anothor part of your website. They are clickable elements on web pages, emails, or documents that connect to other resources

Example HTML Code:

<!DOCTYPE html>
<html lang="en">
<body>
    <p>This is a Normal Link.</p>
<a href="https://www.example.com/">Example Website</a>
     
 <p>This is a Link That Opens In a New Window.</p>		
 
 <a href="https://www.example.com/" target="_blank">Example Website (Opens In New Window.)</a>	

 <p>This is a Email link</p>

 <a href="mailto:fox@example.com">Email daz Fox</a>	
</body>
</html>

Output

This is a Normal Link.

Example Website


This is a Link That Opens In a New Window.


Example Website (Opens In New Window.)
 

This is a Email link

Email daz Fox
