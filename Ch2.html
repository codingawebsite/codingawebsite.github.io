Chapter 2: HTML Text

The Main Way to Show Text in HTML

Like HTML structure, HTML text is also a very, very important (of course). Text is also important because it is a major tool for styling and presentation. There are many ways to show text in HTML like <b> for bold and <i> for italic.

Example HTML Code:

<!DOCTYPE html>
<html lang="en">
<body>
    <h1>This is Big Text</h1>
	<h2>This is a Heading 2</h2>
	<h3>This is a Heading 3</h3>
	<h4>This is a Heading 4</h4>
	<h5>This is a Heading 5</h5>
	<h6>This is Small Text</h6>
    <p>This is a paragraph of content that is visible to users.</p>
	
	
</body>
</html>

Output:



This is Big Text


This is a Heading 2


This is a Heading 3


This is a Heading 4


This is a Heading 5


This is Small Text



This is a paragraph of content that is visible to users.
