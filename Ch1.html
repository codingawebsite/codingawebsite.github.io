Chapter 1: Basic HTML

Every Webpage Uses Basic HTML Structure

Everyday websites like social media platforms, news websites, search engines, and more use basic HTML structure.

Structure is important for making websites easier to read for humans, and softwear (example: search engines and screen readers). It also provides a essential base for any website. This important aspect also helps provide organized content, in a consistent way.

Example HTML Code:

<!DOCTYPE html>
<html lang="en">
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph of content that is visible to users.</p>
</body>
</html>

Output:

This is a Heading


This is a paragraph of content that is visible to users.

Fact: Web files are stored in .HTML or hypertext markup language files (which is the coding language above), CSS. files or cascading style sheets and .JS files or JavaScript
